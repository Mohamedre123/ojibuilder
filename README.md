# oji builder

منصة عربية لبناء المواقع بالذكاء الاصطناعي — اكتب فكرتك، واحصل على موقع كامل، وعدّل أي جزء بالأمر أو يدويًا.

## التشغيل

```bash
npm install
cp .env.local.example .env.local   # ضع مفتاح ANTHROPIC_API_KEY
npm run dev
```

ثم افتح http://localhost:3000

## كيف يعمل

1. **التوليد** — البرومبت يُرسل إلى `/api/generate` ويعيد مستند HTML كامل (Tailwind + RTL) يُعرض فورًا في iframe.
2. **التعديل** — أي طلب تعديل + الكود الحالي يُرسل إلى `/api/edit` ويعيد النسخة المحدّثة.
3. **التحرير اليدوي** — تبويب «الكود» يتيح تعديل HTML مباشرة مع تحديث المعاينة لحظيًا.
4. **التنزيل** — تصدير الموقع كملف `.html` مستقل.

## البنية

- `app/page.tsx` — الواجهة الرئيسية + القوالب الجاهزة
- `app/builder/page.tsx` — مساحة العمل (محادثة + معاينة + كود)
- `app/api/generate` و `app/api/edit` — محرّك الذكاء الاصطناعي
- `lib/prompts.ts` — هندسة الأوامر والقوالب
- `lib/anthropic.ts` — عميل Anthropic واختيار النماذج

## خارطة الطريق (المراحل التالية)

- بثّ مباشر (streaming) لرؤية الموقع وهو يُبنى
- تعديل بصري بالنقر على أي عنصر مباشرة
- تطبيقات full-stack بقاعدة بيانات (Convex/Supabase) — الجزء المكافئ لتطبيقات stunning
- حسابات مستخدمين، حفظ المشاريع، نشر على نطاقات فرعية
